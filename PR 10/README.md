# Практика 10 — Фронтенд + бэкенд

## Запуск

1. **Бэкенд** (порт 3000):
   ```bash
   cd server
   npm install
   npm start
   ```

2. **Фронтенд** (порт 5173):
   ```bash
   cd client
   npm install
   npm run dev
   ```

3. Открыть http://localhost:5173

## Функционал

- Регистрация и вход (JWT + refresh-токены)
- Хранение токенов в localStorage
- Axios interceptors: подстановка accessToken, автообновление при 401
- Управление курсами: список, создание, просмотр, редактирование, удаление
- Защищённые маршруты — доступ только после авторизации
