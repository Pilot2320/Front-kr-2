# Практика 11 — RBAC (роли) + управление пользователями

## Запуск

1. **Бэкенд** (порт 3000):
   ```bash
   cd server
   npm install
   npm start
   ```

2. **Фронтенд** (порт 5173):
   ```bash
   cd client
   npm install
   npm run dev
   ```

3. Открыть http://localhost:5173

## Функционал

- Регистрация и вход (JWT + refresh-токены)
- Хранение токенов в localStorage
- Axios interceptors: подстановка accessToken, автообновление при 401
- RBAC роли: `user`, `seller`, `admin`
- Права доступа к маршрутам по таблице из практики №11
- Управление курсами:
  - `user`: просмотр списка и карточки
  - `seller`: создание/редактирование + просмотр
  - `admin`: всё как `seller` + удаление
- Управление пользователями (только `admin`): список, изменение роли, блокировка
